== Installation ==

1. Upload the plugin files to `/wp-content/plugins/surecrm/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click "Get Started Now" to connect your WordPress site to SureCRM
4. Complete the authentication process to link your workspace

== Frequently Asked Questions ==

= What is SureCRM? =
SureCRM is a cloud-based CRM platform that helps you manage customer relationships and data. This plugin connects your WordPress site to your SureCRM workspace.

= Do I need a SureCRM account? =
Yes, you need an active SureCRM account to use this plugin. Visit https://app.surecrm.com/ to sign up.

= Is my data secure? =
Yes. The plugin uses OAuth-based authentication and encrypts all sensitive credentials. Data transmission occurs over secure HTTPS connections.

= How often does data sync? =
The plugin automatically syncs data daily. You can also manually trigger a sync anytime from the plugin dashboard.

= Can I disconnect my site? =
Yes. You can disconnect your WordPress site from SureCRM at any time from the plugin settings. This will remove all authentication data.

= Is SureCRM compatible with page builders? =
Yes. SureCRM works with all major WordPress page builders including Elementor, Beaver Builder, Divi, and others.

= Does this plugin slow down my site? =
No. The plugin is lightweight and optimized for performance with minimal impact on your WordPress site.
